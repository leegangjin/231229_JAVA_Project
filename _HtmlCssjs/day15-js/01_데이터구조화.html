많은 양의 데이터가 실시간으로 전달되어야할 때 가장 효율적으로 데이터를 보내는 방법 데이터 구조화 => 구조화된 텍스트 =>JSON

  • JSON(JavaScrpit Object Notation) JavaScrpit 객체를 구조화된 데이터로 표현하기 위한 문자기반 표준 포맷
  • 웹 서버(<->) 화면 데이터를 전송할 떄 일반적으로 사용
  • 자바 스크립트의 객체 형태를 가지고 있기 때문에, 구조파악이 쉽고, 제작도 쉽다.
  • 특히 웹 분야의 자바 스크립트가 모든 프론트에 호환된다는 점을 활용 데이터 수집, 출력용으로 거의 모든 플랫폼 서비스에서 사용되고 있음.
